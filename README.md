# Python-Interpreter
Python Interpreter written in C++ for fun

### Building

```bash
  $ git clone https://github.com/BRCode4Fun/Python-Interpreter
  $ cd Python-Interpreter
  $ make
```

### Running Tests

```bash
  $ make test
```
